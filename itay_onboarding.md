# מדריך למצטרף – הגדת אשכנז AI 🍷

אהלן איתי! מוזמן להצטרף לפיתוח של הגדת הפסח האינטראקטיבית. הנה כל מה שאתה צריך כדי להרים את הפרויקט אצלך במחשב ולהתחיל לעבוד.

## 🚀 שלבי התקנה

**שכפול הריפו (Clone):** פתח טרמינל והרץ את הפקודה:

```bash
git clone https://github.com/oren001/Seder_Pesah.git
cd Seder_Pesah
```

**התקנת תלויות:** וודא שמותקן אצלך Node.js, ואז הרץ:

```bash
npm install
```

**הגדרת משתני סביבה:** צור קובץ בשם `.env` בשורש הפרויקט והוסף את המפתח של Leonardo AI (בקש מאורן אם אין לך):

```env
LEONARDO_API_KEY=your_key_here
PORT=3001
```

**הרצה מקומית:** להרצת השרת:

```bash
node server.js
```

האתר יהיה זמין בכתובת: http://localhost:3001

## 🛠 מבנה הפרויקט

- `server.js`: לב הפרויקט. שרת Express וסנכרון בזמן אמת (Socket.io).
- `leonardo.js`: האינטגרציה עם הבינה המלאכותית ליצירת תמונות.
- `public/`: כל הפרונטנד (HTML, CSS, JS).
- `public/haggadah_data.js`: מכיל את טקסט ההגדה בפורמט JSON.
- `public/app.js`: הלוגיקה של צד הלקוח.

## 🚢 עבודה ודחיפה (Push)

לפני שאתה מתחיל לעבוד, תמיד כדאי למשוך את הגרסה האחרונה:

```bash
git pull origin master
```

אחרי שסיימת שינוי:

```bash
git add .
git commit -m "תיאור קצר של מה שעשית"
git push origin master
```

## 🌍 פריסה לשרת (Deployment)

השרת שלנו נמצא בכתובת 45.77.50.140. אם אתה עושה שינוי שדורש עדכון בשרת, הפעולה פשוטה:

1. מתחברים ב-SSH לשרת.
2. רצים:
```bash
cd Seder_Pesah
./deploy.sh
```

(הסקריפט מטפל בבנייה מחדש של ה-Docker וב-SSL).
