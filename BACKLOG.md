## 🏗 ארכיטקטורה נוכחית

```
server.js          ← Express + Socket.io (backend)
leonardo.js        ← AI image generation (Leonardo Phoenix)
public/
  app.js           ← Client logic (socket + UI)
  index.html       ← Single page app
  styles.css       ← Dark theme + glassmorphism
  haggadah_data.js ← כל טקסט ההגדה (Hebrew + transliteration)
  ExodusMap.js     ← רכיב מפת הסדר
  RSVPFlow.js      ← רכיב הצטרפות רב-שלבי
  sw.js            ← Service Worker (PWA)
data/
  rooms.json       ← חדרים (persistence)
  selfies.json     ← תמונות משתמשים
  tasks.json       ← משימות
```
